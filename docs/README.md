# Documentação do Projeto EasyStop

Bem-vindo à documentação do EasyStop - o sistema que transforma o caos do estacionamento em ordem digital. Esta pasta é o repositório oficial de todo o conhecimento técnico e de negócios que alimenta nossa aplicação.

Pense nisso como o manual de sobrevivência para desenvolvedores, testadores e arquitetos que precisam entender o sistema sem precisar ler o código fonte linha por linha (embora, sejamos honestos, você provavelmente vai precisar fazer isso também).

-----------------

# O que você vai encontrar aqui?

Documentação é como cebola: tem camadas. E também pode fazer você chorar se estiver mal feita. Felizmente, organizamos a nossa em camadas lógicas:

O "Porquê" - Documentos de visão e requisitos
O "O quê" - Casos de uso e processos de negócio
O "Como" - Arquitetura e modelagens técnicas
O "Com o quê" - Boas práticas e padrões

-----------------
